How To

To use it, you will need to do 2 things:
Find the api library for your OS, basically:
  • Linux: libEOSSDK-Linux-Shipping.so
  • MacOSX: libEOSSDK-Mac-Shipping.dylib
  • Windows: EOSSDK-Win32-Shipping.dll or EOSSDK-Win64-Shipping.dll

Make a backup and replace it with mine.
Create the emulator settings next to your game's executable. Some games use a launcher (not the epic game launcher, an intermediate launcher) to start the game, so you must find what is the final executable, not the launcher.
For example, on Satisfactory, there is a FactoryGame.exe, thats the launcher, the game is located in FactoryGame/Binaries/Win64, thats where you must create the emulator config.


An example config file can be found here: NemirtingasEpicEmu.json

If you don't provide an epicid or if your currently configured epicid is invalid, it will be regenerated with your username as seed.
If you want to have an unique epicid, remove the epicid from the .json and setup your username. The generated id will be identical for the same username.
