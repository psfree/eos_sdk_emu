You will need .NET Core 3.1 to run this.

You can download .NET Core on https://dotnet.microsoft.com/download/dotnet-core/3.1

Download the .NET Core Runtime for your OS (installer recommanded) and run the launcher depending on your OS:
Windows = NemirtingasEmuLauncher.exe
Linux = NemirtingasEmuLauncher.linux
MacOS = NemirtingasEmuLauncher.macos

If .NET Core is installed, it will start, else if you downloaded the Binaries, call the "dotnet" executable with NemirtingasEmuLauncher.dll as a parameter.
